# Resume Agent - AI Resume Analyzer

Live Demo: https://your-frontend.onrender.com  
API Docs: https://your-backend.onrender.com/docs  

## Tech Stack
- React (Vite)
- FastAPI
- LangChain + Groq + Tavily
- Render (Deployment)

## Features
- AI resume analysis
- Market skill comparison
- Structured JSON output
- Real-time backend logs

## Architecture
React → FastAPI → LangChain Agent → LLM

## Run locally

Backend:
uvicorn main:app --reload

Frontend:
npm run dev
